<span data-ttu-id="c3044-147">Focalizzando l'attenzione su semplicità ed espressività, con Markdown si è riusciti a creare una descrizione coerente e semplice del contenuto del documento.</span><span class="sxs-lookup"><span data-stu-id="c3044-147">By focusing on keeping it simple and expressive, Markdown created an easy and consistent description of document content.</span></span>  <span data-ttu-id="c3044-148">Nello stesso modo, Microsoft ritiene che le schede adattive possano diventare uno strumento semplice ed espressivo per descrivere il contenuto della scheda.</span><span class="sxs-lookup"><span data-stu-id="c3044-148">In the same way, we believe that Adaptive Cards can create a simple, expressive means of describing card content.</span></span>
